pub fn pow<T>(base: T, exp: usize) -> T where
    T: One<Output = T> + Mul<T> + Clone, 

Raises a value to the power of exp, using exponentiation by squaring.

Example

use num_traits::pow;

assert_eq!(pow(2i8, 4), 16);
assert_eq!(pow(6u8, 3), 216);Run
