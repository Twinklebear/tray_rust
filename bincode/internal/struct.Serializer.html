pub struct Serializer<W, E: ByteOrder> { /* fields omitted */ }

An Serializer that encodes values directly into a Writer.

The specified byte-order will impact the endianness that is used during the encoding.

This struct should not be used often. For most cases, prefer the encode_into function.
